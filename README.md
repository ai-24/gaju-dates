# Gaju-dates

This npm show you Gaju dates that are celebrated longevity in Japan when you answer two questions.

## Install

Install with npm:
```
$ npm install gaju-dates
```

## Usage

1. Enter your birthday. (yyyy/mm/dd)
   
<img src="https://user-images.githubusercontent.com/76685187/156134522-5a6ee157-2754-4451-b3c0-7d3ccfa4ba89.png" width=70%>
   
2. Enter your name.
   
   <img src="https://user-images.githubusercontent.com/76685187/156134755-52aabc92-4a77-4bf2-844a-d0210ebbedde.png" width=55%>
   
3. If you want to know another person's one, you can enter y and then repeat 1 and 2. If not, enter n.
